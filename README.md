# PdfjsRails

Gem gem packages PDF.js from Mozilla for rails asset pipeline.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pdfjs_rails'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pdfjs_rails

## Usage

Include assets with

    //= require pdfjs/l10n
